Project Description

Smart Budget Tracker Lite is a mobile application developed using React Native, Expo, and TypeScript. The app allows users to add income and expense transactions, view totals, and manage data using AsyncStorage. It demonstrates multi-screen navigation, basic state management, and clean UI design as part of the MAD201 course project.

Features
- Add income and expense transactions  
- View and delete transactions  
- Display total income, expenses, and balance  
- Persistent storage using AsyncStorage  
- Multiple screens with Expo Router navigation  
- Simple and responsive user interface

Technologies Used
- React Native  
- Expo  
- TypeScript  
- AsyncStorage  
- Expo Router

How to Run the Project
1. Install dependencies:
   npm install
2. Start the application:
   npx expo start
3. Run the app using an emulator or Expo Go.
